# AI Prompt Engineering Hub
A centralized repository for AI prompt engineering, featuring a curated collection of viral AI image prompts, ranked by likes and updated weekly.

## Description
This hub provides a platform for developers and researchers to share, discover, and collaborate on prompt engineering techniques, including system prompt leaks and jailbreak methods.

## Features
* Fetch viral AI image prompts from a remote repository
* Rank prompts by likes
* Save prompts to a file

## Installation
To install the AI Prompt Engineering Hub, run the following command:
```bash
pip install -r requirements.txt
```

## Usage
To fetch viral prompts, rank them, and save them to a file, run the following command:
```bash
python main.py --fetch --rank --save --filename prompts.txt
```
To fetch viral prompts and save them to a file without ranking, run the following command:
```bash
python main.py --fetch --save --filename prompts.txt
```
To rank prompts and save them to a file without fetching, run the following command:
```bash
python main.py --rank --save --filename prompts.txt
```
